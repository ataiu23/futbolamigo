# futbolamigo

A Flutter project for football/sports management.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio or Xcode (for iOS)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/ataiu23/futbolamigo.git
   ```

2. Install dependencies:
   ```
   flutter pub get
   ```

3. Run the app:
   ```
   flutter run
   ```

### Project Structure

- `lib/` - Main source code
- `lib/perfilpage/` - User profile page
- `lib/partidopage/` - Match page
- `lib/notificacionpage/` - Notifications page
- `lib/resultadospage/` - Results page
- `lib/pdf_viewer/` - PDF viewer functionality
- `lib/selectpageequipos/` - Team selection page
